HTML en CSS:

HTML (HyperText Markup Language):

Dit is een taal om webpagina's te maken. Het is als een bouwplan voor een website. Hiermee plaats je tekst, afbeeldingen en knoppen op een webpagina.

CSS (Cascading Style Sheets):

Dit is een taal die helpt bij het mooi maken van een website. Het bepaalt kleuren, lettertypen en hoe dingen eruitzien op een webpagina.

Kort gezegd, software zijn handige programma's op de computer. HTML en CSS zijn talen om websites te maken en mooi te laten uitzien. HTML maakt de structuur en CSS zorgt voor de mooie opmaak.
